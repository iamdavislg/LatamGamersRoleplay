
LatamGamers Roleplay - Plantilla Web
===================================

Archivos incluidos:
- index.html
- style.css
- script.js
- images/LatamGamersRoleplay.png (tu logo)
- images/gallery1.png (imagen de galería)

Cómo subir a GitHub Pages (rápido):
1. Crea una cuenta en https://github.com si no tienes una.
2. Crea un nuevo repositorio público (por ejemplo: latamgamers-roleplay).
3. Haz clic en "Add file" -> "Upload files". Arrastra todos los archivos y la carpeta 'images'.
4. Pulsa "Commit changes".
5. Ve a Settings -> Pages -> Source -> selecciona branch 'main' y carpeta '/ (root)'. Guarda.
6. En minutos, tu web estará en https://tuusuario.github.io/nombre-del-repo

Editar textos e imágenes:
- Abre index.html con un editor de texto y modifica el contenido que necesites.
- Reemplaza images/LatamGamersRoleplay.png o images/gallery1.png por otras imágenes si lo deseas.
